### 📱 訪問方式

#### 方法 1：在 Jetson 本機瀏覽器訪問
```
http://localhost:5000
```

#### 方法 2：從其他設備訪問（手機、平板、筆電）
確保您的設備與 Jetson 在同一個區域網路（Wi-Fi），然後在瀏覽器中打開：
```
http://192.168.0.154:5000
```

### 🎯 使用步驟

1. **打開瀏覽器**
   - 推薦使用 Chrome、Firefox 或 Safari
   
2. **輸入網址**
   - 輸入上面的任一網址

3. **查看畫面**
   - 左側：實時攝像頭畫面
   - 右側：辨識結果和手勢說明

4. **開始辨識**
   - 將手放在攝像頭前方
   - 比出 0-5 的手勢
   - 保持穩定約 0.5 秒
   - 查看右側顯示的數字結果

### 📖 手勢參考

| 數字 | 手勢 | 說明 |
|------|------|------|
| 0 | ✊ | 握拳 |
| 1 | ☝️ | 伸出一根手指 |
| 2 | ✌️ | 剪刀手 |
| 3 | 🤟 | 伸出三根手指 |
| 4 | 🖖 | 伸出四根手指 |

### 💡 提示

- 確保光線充足
- 手部盡量放在畫面中央
- 手勢要清晰明確
- 如果辨識不準，可以調整手與攝像頭的距離

### 🐛 常見問題

**Q: 網頁打不開？**
- 檢查伺服器是否在運行
- 確認防火牆沒有阻擋 5000 端口
- 確認設備在同一網路

**Q: 看不到攝像頭畫面？**
- 檢查攝像頭是否正確連接
- 確認攝像頭驅動已安裝
- 查看終端是否有錯誤訊息

**Q: 辨識不準確？**
- 改善光線條件
- 調整手與攝像頭距離
- 保持手勢穩定時間更長
